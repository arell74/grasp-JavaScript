Boolean

boolean hanya bernilai true atau false

contoh : "Farel Fauzan" == "farel fauzan"

maka hasilnya akan false

pada Javascript sebenarnya semua nilai itu menyimpan nilai true di dalamnya(truthy | falsy)

// semua nilai di Javascript itu menyimpan boolean didalamnya

misal :

Boolean("farel") hasilnya true

Boolean(10) hasilnya true

Boolean(0) hasilnya false

Boolean(-10) hasilnya true
