Variable

Variable adalah suatu wadah yang memiliki nama, yang digunakan untuk menyimpan nilai

tahap tahap variable

  1. Deklarasi
    2. mendaftarkan variable kedalam lingkup yang sesuai
  2. Inisialisasi
    3. menyediakan memori untuk variable
  3. Assigment
    4. menetapkan nilai spesifik ke dalam variable
contoh

var x; //ini tuh sudah melakukan deklarasi dan inisialisasi

x = 20; //ini adalah Assigment

deklarasi variable
  • var
  • let
  • const
var (nama variable)

Rules!

var gaboleh pake spasi;

var boleh1; //var boleh mengandung angka

var 1gaboleh; //tidak boleh diawali dengan angka

var ini_juga_boleh;

var boleh$;

var $_$;
