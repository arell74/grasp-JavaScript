Logika, string, Typeof dan Kondisional



Operator Logika

ketika ada kondisi && (and) itu dua duanya harus true untuk hasilkan true,

kalau yang kiri true yang kanan false maka hasilnya pasti false

(x % 2 == 0) {ini hasilnya false. x nya 5}

(x % 2 == 0) && (x < 10) {ini hasilnya false}


=================================================

ketika ada kondisi || (or) itu syaratnya jika salah satunya true, maka hasilnya true

kalau yang kiri true dan yang kanan false maka hasilnya pasti true

(x % 2 == 0) || (x < 10) {ini hasilnya false}


=================================================

ketika ada kondisi ! () itu syaratnya jika nilainya true, maka hasilnya false, dibalik

!(x < 10) {ini hasilnya false}



Operator String

Di Javascript, ada operator + yang mana fungsinya bisa 2. tergantung operand nya, jika operandnya angka maka akan menjadi perjumlahan, tapi jika operand nya string maka akan menjadi penggabung kata

contoh

a = "apple"

b = "pen"

a + b = applepen

*jika ingin ada jarak maka tinggal tambah saja jadi a + " " + b = apple pen


Jika angka dengan angka, maka hasilnya juga angka, tapi,

10 + 10 = 20

10 + "10" = 1010

10 + 10 + "10" = 2010

"10" + 10 + 10 = 101010



Operator Typeof

hanya butuh 1 operand untuk mengetahui type data

typeof(10) maka hasilnya number

typeof("10") maka hasilnya string

typeof(true) maka hasilnya boolean



Operator Kondisional

(kondisi) ? benar : salah

(x % 2 == 0) ? "genap" : "ganjil"
