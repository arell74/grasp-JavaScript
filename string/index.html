String di Javascript

penggunaan string di Javascript bisa ditandai dengan kutip satu('') atau kutip dua("")

penggunaannya bebas tergantung kondisi, misal

"hari ini adalah hari jum'at"

'kami sedang mengadakan kegiatan "Gerakan pungut sampah"'

akan ada kondisi dimana keduanya akan kepakai, maka cara mengakalinya dengan

memakai escape carachter

===== contoh =====

'"Gerakan pungut sampah" dilaksanakan pada hari jum\'at'

"\"Gerakan pungut sampah\" dilaksanakan pada hari jum'at"


===== contoh escape carachter lainnya =====

\0 = karakter null

\' = '

\" = "

\\ = \

\n = newline / garis baru

\t = tab

\b = backspace

\uXXXX = unicode (\u00A9)

*jika ingin mengetes pastikan pakai cara console.log('testing \nGaris baru')

.length

untuk mengukur panjang dari sebuah string.

misal : "farel".length
